My Projects

Number Guessing Game

This Number Guessing Game is an interactive web-based game where players try to guess a randomly generated number between 1 and 100 within a set number of attempts. The game provides feedback after each guess, telling the player whether the guess is too high or too low. The player has 5 attempts to guess the correct number, and the game tracks the number of attempts used. It is a fun and engaging way for players to improve their logical thinking skills.

Key Features:

  • Random number generation between 1 and 100
  • Feedback on whether the guess is too high or too low
  • 5 attempts to guess the correct number
  • Play Again option after each round

ATM Interface Application (Java)

The ATM Interface project is a command-line Java application that simulates basic ATM functionalities such as withdrawing money, depositing money, checking account balance, and exiting the session. The system is PIN-protected and built with clean, modular code structure to ensure reusability and scalability.

Key Features:

  • User authentication with PIN verification
  • Options for Withdraw, Deposit, and Check Balance
  • Error-handling for invalid inputs and insufficient balance
  • Object-oriented design using classes like ATMSystem and UserAccount
